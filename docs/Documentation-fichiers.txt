sources:
    main.py:
        Contient la boucle principale du jeu, la gestion d'affichage des images et de dessin à l'écran ainsi que la fonction de démarrage du jeu.
    
    IA_Ennemis.py:
        Contient le code des agents (IA) des ennemis (Basiques, Chasseurs, Intelligents)
    
    Navire.py:
        Contient la classe Navire, contenant toutes les méthodes liées aux navires (joueur et ennemis)
    
    class_button.py:
        Contient le code des boutons du menu de l'interface principale
    
    class_menu.py:
        Contient le code du menu de l'interface principale
    
    iles.py:
        Contient le code des îles (génération, récompense)
    
    ressources.py:
        Contient les diverses ressources nécessaires au bon fonctionnement du code (comparaisons, listes importantes…)
    
    shot.py:
        Contient le code des tirs des canons
data:
    Fonts:
        pixel-font.otf:
            Fichier de police d'écriture

    images:
        Backgrounds:
            Contient les frames du gif de fond d'écran

        Icons:
            Contient les différentes icones du jeu (bénédictions, canons, coques, voiles)

        Interfaces:
            Contient les différentes images des interfaces du jeu (barre de vie, Interface des bénédictions, Interface des équipements)
        
        Textures:
            Contient les différentes textures du jeu (bateaux, îles)

        README:
            Contient les icones à afficher dans le README

docs:
    Contient différents fichiers de documentation (arborescence, licence, présentation)
